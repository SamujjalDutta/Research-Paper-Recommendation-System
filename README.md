# Research Paper Recommendation System

A full-stack web application that recommends research papers based on topic similarity using NLP techniques.

## Tech Stack
Frontend: React  
Backend: Python (Flask)  
Machine Learning: TF-IDF + Cosine Similarity

## Features
- Search research topic
- Get recommended research papers
- Minimal and responsive UI

## Project Status
Frontend implemented.  
Backend recommendation model under development.
